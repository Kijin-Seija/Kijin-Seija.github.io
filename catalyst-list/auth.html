# 权限管理

在表格组件的field配置，以及表单组件和交互按钮组件的配置中，都有这样两项参数：

authority_code 权限码，如果不配置默认公开权限

show_condition 显示状态控制，如果不配置默认为true。

这两项配置无论在任何地方，最终都由权限控制系统处理。

# authority_code

要对接权限管理功能，我们首先需要在安装Catalyst的时候在全局配置一个权限接口方法hasPermit。

hasPermit接受一个参数authorityCode，该参数即为组件的权限码。开发者可以自己实现基于权限码的方法，以判断是否显示该组件

# show_conditions

该参数扩展了权限功能，允许传入一段自定义的js函数字符串，该函数必须返回一个布尔值，以确定是否显示该组件。

一个组件必须 hasPermit（authority_code) 和 show_condition 均为true, 才会显示
